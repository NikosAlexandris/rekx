    app¶

    🙾 🦖 Rekx command line interface [bold][magenta]prototype[/magenta][/bold]

    Usage:

    callback-app [OPTIONS] COMMAND [ARGS]...

    Options:

      --version                       Show the version and exit.
  --install-completion [bash|zsh|fish|powershell|pwsh]
                                  Install completion for the specified shell.
  --show-completion [bash|zsh|fish|powershell|pwsh]
                                  Show completion for the specified shell, to
                                  copy it or customize the installation.

    combine¶

    Combine Kerchunk reference sets (JSONs to JSON)

    Usage:

    callback-app combine [OPTIONS] SOURCE_DIRECTORY [COMBINED_REFERENCE]

    Options:

      SOURCE_DIRECTORY          Source directory path  [required]
  --pattern TEXT            Filename pattern to match  [default: *.json]
  [COMBINED_REFERENCE]      Combined kerchunk reference file name  [default:
                            combined_kerchunk.json]
  --dry-run / --no-dry-run  Run the command without making any changes.
                            [yellow bold reverse] Try Me! [/yellow bold
                            reverse]  [default: no-dry-run]
  -v, --verbose             Show details while executing commands  [default:
                            0]

    inspect¶

    Inspect Xarray-supported data

    Usage:

    callback-app inspect [OPTIONS] [INPUT_PATH] [LONGITUDE] [LATITUDE]

    Options:

      [INPUT_PATH]                    Input filename or directory path  [default:
                                  .]
  --pattern TEXT                  Filename pattern to match  [default: *.nc]
  --variable TEXT
  --variable-set [all|coordinates|coordinates-without-data|data|metadata|time]
                                  Set of Xarray variables to diagnose
                                  [default: XarrayVariableSet.all]
  --long-table / --no-long-table  [default: long-table]
  --group-metadata / --no-group-metadata
                                  [default: no-group-metadata]
  [LONGITUDE]                     Longitude in decimal degrees ranging in
                                  [-180, 360]. [yellow]If ranging in [0, 360],
                                  consider the `--convert-longitude-360`
                                  option.[/yellow]  [default: 8]
  [LATITUDE]                      Latitude in decimal degrees ranging in [-90,
                                  90]  [default: 45]
  --repetitions INTEGER           Times to repeat the reading operation and
                                  calculate the average read time.  [default:
                                  10]
  -h, --humanize                  Convert byte sizes into human-readable
                                  formats
  --csv PATH                      CSV output filename
  -v, --verbose                   Show details while executing commands
                                  [default: 0]

    rechunk¶

    Rechunk data

    Usage:

    callback-app rechunk [OPTIONS] INPUT OUTPUT_DIRECTORY

    Options:

      INPUT                           Input NetCDF file.  [required]
  OUTPUT_DIRECTORY                Path to the output NetCDF file.  [required]
  --time INTEGER                  New chunk size for the `time` dimension.
                                  [required]
  --latitude INTEGER              New chunk size for the `lat` dimension.
                                  [required]
  --longitude INTEGER             New chunk size for the `lon` dimension.
                                  [required]
  --variable-set [all|coordinates|coordinates-without-data|data|metadata|time]
                                  Set of Xarray variables to diagnose
                                  [default: XarrayVariableSet.all]
  --cache-size INTEGER            [default: 16777216]
  --cache-elements INTEGER        [default: 4133]
  --cache-preemption FLOAT        [default: 0.75]
  --compression TEXT              [default: zlib]
  --compression-level INTEGER     [default: 4]
  --shuffling TEXT
  --memory / --no-memory          [default: no-memory]
  --dry-run / --no-dry-run        Run the command without making any changes.
                                  [yellow bold reverse] Try Me! [/yellow bold
                                  reverse]  [default: dry-run]
  --backend [all|netCDF4|xarray|nccopy]
                                  Backend to use for rechunking.
                                  [code]nccopy[/code] [red]Not Implemented
                                  Yet![/red]  [default:
                                  RechunkingBackend.nccopy]
  --dask-scheduler TEXT           The port:ip of the dask scheduler
  -v, --verbose                   Show details while executing commands
                                  [default: 0]

    select¶

     Select time series over a location

    Usage:

    callback-app select [OPTIONS] TIME_SERIES VARIABLE LONGITUDE LATITUDE [TIMESTAMPS]

    Options:

      TIME_SERIES                     [required]
  VARIABLE                        Variable name to select from  [required]
  LONGITUDE                       Longitude in decimal degrees ranging in
                                  [-180, 360]. [yellow]If ranging in [0, 360],
                                  consider the `--convert-longitude-360`
                                  option.[/yellow]  [required]
  LATITUDE                        Latitude in decimal degrees ranging in [-90,
                                  90]  [required]
  --list-variables / --no-list-variables
                                  List variables in the Kerchunk reference set
                                  [default: no-list-variables]
  [TIMESTAMPS]                    Quoted date-time strings of data to extract
                                  from series, example: [yellow]'2112-12-21,
                                  2112-12-21 12:21:21, 2112-12-21
                                  21:12:12'[/yellow]'
  --start-time [%Y-%m-%d|%Y-%m-%dT%H:%M:%S|%Y-%m-%d %H:%M:%S]
                                  Start timestamp of the period.
                                  [yellow]Overrides the `timestamps`
                                  paramter![/yellow]
  --end-time [%Y-%m-%d|%Y-%m-%dT%H:%M:%S|%Y-%m-%d %H:%M:%S]
                                  End timestamp of the period.
                                  [yellow]Overrides the `timestamps`
                                  paramter![/yellow]
  --time INTEGER                  New chunk size for the 'time' dimension
  --lat INTEGER                   New chunk size for the 'lat' dimension
  --lon INTEGER                   New chunk size for the 'lon' dimension
  --mask-and-scale / --no-mask-and-scale
                                  Mask and scale the series  [default: no-
                                  mask-and-scale]
  --neighbor-lookup [None|pad|backfill|nearest]
                                  Enable nearest neighbor (inexact) lookups.
                                  Read Xarray manual on [underline]nearest-
                                  neighbor-lookups[/underline]  [default:
                                  MethodForInexactMatches.nearest]
  --tolerance FLOAT               Maximum distance between original & new
                                  labels for inexact matches. Read Xarray
                                  manual on [underline]nearest-neighbor-
                                  lookups[/underline]  [default: 0.1]
  --in-memory / --no-in-memory    Use in-memory processing  [default: no-in-
                                  memory]
  --statistics / --no-statistics  Calculate and display summary statistics
                                  [default: no-statistics]
  --csv PATH                      CSV output filename
  --variable-name-as-suffix / --no-variable-name-as-suffix
                                  Suffix the output filename with the variable
                                  name  [default: variable-name-as-suffix]
  -r, --rounding-places INTEGER   Number of digits to round results to
                                  [default: 5]
  -v, --verbose                   Show details while executing commands
                                  [default: 0]
